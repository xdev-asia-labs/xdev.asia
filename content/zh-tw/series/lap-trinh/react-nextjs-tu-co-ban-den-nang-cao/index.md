課程 **React 和 Next.js：從基礎知識到高級** 幫助您掌握 React 19+ 和 Next.js 15+ - 當今最受歡迎的前端/全端組合，從基本元件到生產就緒的全端應用程式。

## 你會學到什麼？

- **React Core**：元件、掛鉤、狀態管理、模式、效能
- **Next.js**：應用程式路由器、伺服器元件、伺服器操作、資料獲取
- **全端**：身份驗證、資料庫（Prisma）、檔案上傳、即時、支付
- **生產**：測試、Docker、CI/CD、Vercel 部署、監控

## 請求

- HTML、CSS、JavaScript ES6+ 的基本知識
- 對 TypeScript 有基本了解（建議）
- Node.js 20+ 和 npm/pnpm
