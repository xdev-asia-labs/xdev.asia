技術上出色但無法說服利害關係人的BA = 專案失敗。擅長所有事但無法面試的BA = 沒有機會證明自己。AI模擬是有回饋、免費且不限次數地練習的方式。

---

## 1. AI的利害關係人模擬

### 1.1 為什麼模擬有效？

困難的利害關係人有重複的模式：
- **懷疑者：** 「我們真的需要AI嗎？手動難道不夠嗎？」
- **過度要求者：** 「把這個功能也加上——應該不需要太久吧？」
- **風險規避者：** 「如果AI出錯怎麼辦？我不會簽字的。」
- **預算守門員：** 「這個ROI確定嗎？計算根據是什麼？」
- **技術擁護者：** 想讓AI做所有事、不肯聽使用者需求

用AI模擬練習 → 識別模式 → 當真正遇到時更有效地處理。

### 1.2 利害關係人模擬提示模板

**設置提示：**
```
你是[利害關係人名稱]，[X行業公司]的[職位]。
你的特徵如下：
- [特徵1：例：「總是在承諾前詢問ROI和數字」]
- [特徵2：例：「因為之前的AI專案失敗而對AI持懷疑態度」]
- [特徵3：例：「有預算權限但不懂技術」]

我（BA）將向你展示一個AI功能提案。
按照你的角色反應，提出尖銳的問題，必要時提出異議。
開場白：「好，我有20分鐘。告訴我這個功能是什麼。」
```

**情境範例：**
```
情境：我需要說服CFO批准一個AI聊天機器人5萬美元的預算。
CFO之前以「ROI不明確」拒絕了2個AI提案。
我有15分鐘和5張投影片。
```

### 1.3 模擬會議類型

**類型1：探索會議**
```
情境：新的利害關係人，第一次會議要提取需求。
目標：學習如何開始對話、問開放式問題、
       積極傾聽、避免引導式問題。
```

**類型2：異議會議**
```
情境：利害關係人不同意你的需求優先順序。
目標：學習如何處理反對意見、保持冷靜、
       不失去立場地進行談判。
```

**類型3：主管簡報**
```
情境：向總監進行10分鐘的上線批准簡報。
目標：清晰度、信心、應對尖銳的問答。
```

## 2. AI模擬面試

### 2.1 BA的面試類型和模擬

**類型1：行為面試（STAR格式）**

```
給AI的提示：
「請面試我一個專注於AI的資深業務分析師職位。
問我5個行為問題。每個問題之後：
1. 提出問題
2. 我回答後，提供具體回饋：
   - 回答的優點
   - 弱點或缺漏
   - STAR格式的改進建議」
```

**類型2：案例研究面試**

```
給AI的提示：
「向我提出一個BA案例研究：
 一家金融科技公司想用AI自動化信用審查。
 像困難的面試官一樣提問，
 評估問題的結構化、需求識別、
 以及處理模糊性的方式。」
```

**類型3：技術深度檢查**

```
給AI的提示：
「測試我的AI相關BA知識。依序問我：
1. 精確度和召回率的差異——各自何時使用？
2. BA如何為AI模型設計測試集？
3. 為什麼需要人在迴路中（human-in-the-loop）？
4. GDPR第22條如何影響AI功能的需求？
每個回答後給1到5分並說明原因。」
```

## 3. 回饋迴圈：讓模擬真正有價值

### 3.1 每次模擬會議後

```
要求回饋的提示：
「現在請離開角色。
評估我在剛才模擬中的表現：
1. 最好的回答或論述是哪些？為什麼？
2. 哪裡很弱或有所欠缺？
3. 有我沒有準備好的利害關係人問題嗎？
4. 下次需要改進的3件事？」
```

### 3.2 進度追蹤模板

```markdown
## 模擬記錄

| 日期 | 類型 | 情境 | 分數（1到5） | 主要弱點 | 下次改進了？ |
|------|------|----------|-------------|--------------|---------------------|
| YYYY-MM-DD | 利害關係人 | CFO預算批准 | 3/5 | 缺少具體ROI數字 | ☐ |
| YYYY-MM-DD | 模擬面試 | 行為——處理衝突 | 4/5 | STAR結構薄弱 | ☐ |
```

## 4. 進階：用AI角色扮演逐步提升難度

### 逐步增加難度

**等級1：** AI扮演合作和支持型的利害關係人  
**等級2：** AI扮演懷疑但理性的利害關係人  
**等級3：** AI扮演有自己利益的對抗性利害關係人  
**等級4：** AI同時扮演多個利害關係人（利益衝突）

```
等級4提示：
「在同一個會議中扮演2個利害關係人：
- A：CTO，想要盡快上線AI功能，不在乎業務案例
- B：CFO，要求明確的ROI，對AI持懷疑態度

兩人之間存在動態緊張關係。我必須說服兩者。
開始會議。」
```

## 結論

AI模擬不是真實經驗的替代——但它能**加速**準備。10次AI模擬會議 + 1次真實利害關係人會議 = 遠好過只有1次真實會議的結果。

練習困難的問題直到**感到無聊**為止——那就是你準備好上場的信號。
