## Giới thiệu

**xClaw** trên App Store là ứng dụng chat AI cho macOS, mang đến trải nghiệm trò chuyện với nhiều mô hình AI khác nhau trong một giao diện đẹp mắt với thiết kế Liquid Glass.

## Tải về

Tải xClaw miễn phí trên [App Store](https://apps.apple.com).

## Tính năng

### 🤖 Đa mô hình AI
Hỗ trợ GPT-4, GPT-4o, Claude 3.5, Gemini Pro và nhiều mô hình khác. Chuyển đổi giữa các mô hình chỉ với một click.

### 💬 Quản lý hội thoại
- Tạo nhiều cuộc trò chuyện
- Tìm kiếm trong lịch sử
- Ghim cuộc trò chuyện quan trọng
- Xuất hội thoại ra Markdown

### 🎨 Thiết kế Liquid Glass
Giao diện trong suốt, hiệu ứng kính mờ premium theo phong cách Apple. Hỗ trợ cả Dark Mode và Light Mode.

### 📋 Markdown & Code
- Syntax highlighting cho 100+ ngôn ngữ
- Copy code block với một click
- Render bảng, danh sách, hình ảnh
- LaTeX math rendering

## Yêu cầu

- macOS 14.0 (Sonoma) trở lên
- Apple Silicon hoặc Intel Mac

## Hướng dẫn sử dụng

### Bắt đầu

1. Tải ứng dụng từ App Store
2. Mở ứng dụng và vào **Settings** (⌘,)
3. Nhập API key từ provider bạn muốn sử dụng
4. Quay lại màn hình chính và bắt đầu trò chuyện

### Keyboard Shortcuts

| Phím tắt | Hành động |
|----------|-----------|
| `⌘N` | Tạo cuộc trò chuyện mới |
| `⌘,` | Mở Settings |
| `⌘K` | Tìm kiếm cuộc trò chuyện |
| `⌘⇧C` | Copy tin nhắn cuối |
| `Enter` | Gửi tin nhắn |
| `⇧Enter` | Xuống dòng |

## Quyền riêng tư

- API keys được lưu trữ an toàn trong macOS Keychain
- Không thu thập dữ liệu cá nhân
- Lịch sử trò chuyện chỉ lưu trên thiết bị (Core Data)
