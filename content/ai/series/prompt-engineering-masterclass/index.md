## Giới thiệu Series

**Prompt Engineering Masterclass** là khóa học giúp bạn **thành thạo kỹ năng quan trọng nhất trong thời đại AI** — nghệ thuật giao tiếp hiệu quả với các mô hình ngôn ngữ lớn (LLM) như ChatGPT, Claude, Gemini.

> 🎯 **Sự thật:** 80% giá trị bạn nhận được từ AI phụ thuộc vào cách bạn **hỏi**, không phải mô hình nào bạn dùng. Prompt Engineering là kỹ năng biến AI từ "hữu ích" thành "vũ khí bí mật".

## Bạn sẽ học được gì?

### Phần 1: Nền tảng Prompt Engineering

- **Bài 1:** Prompt Engineering là gì? Cấu trúc prompt tối ưu, nguyên tắc CLEAR
- **Bài 2:** System Prompts & Persona Design — tạo AI assistant chuyên nghiệp
- **Bài 3:** Few-Shot, Zero-Shot — khi nào cần ví dụ, khi nào không

### Phần 2: Kỹ thuật Nâng cao

- **Bài 4:** Chain-of-Thought (CoT) — bắt AI giải thích suy luận từng bước
- **Bài 5:** Tree-of-Thoughts, Self-Consistency — explore nhiều hướng giải quyết
- **Bài 6:** Structured Output — JSON mode, schema validation, integration

### Phần 3: Ứng dụng Thực tế

- **Bài 7:** Prompt cho Code Generation — GitHub Copilot, Cursor, Claude Code
- **Bài 8:** Prompt cho Data Analysis & Business Writing
- **Bài 9:** Multimodal Prompting — Image, Audio, Video

### Phần 4: Production & Best Practices

- **Bài 10:** Prompt Testing & Evaluation Framework
- **Bài 11:** Prompt Versioning, A/B Testing trong production
- **Bài 12:** Capstone: Xây Prompt Library cho doanh nghiệp

## Ai nên học khóa này?

| Đối tượng | Lý do |
|-----------|-------|
| **Developer** | Viết prompt tối ưu cho code generation, debugging, testing |
| **Data Analyst** | Phân tích dữ liệu, tạo báo cáo, SQL generation |
| **Product Manager** | Viết spec, PRD, user stories bằng AI |
| **Content Creator** | Tạo content chất lượng cao, đa dạng format |
| **Bất kỳ ai dùng AI** | Tiết kiệm 50%+ thời gian khi tương tác với ChatGPT/Claude |
