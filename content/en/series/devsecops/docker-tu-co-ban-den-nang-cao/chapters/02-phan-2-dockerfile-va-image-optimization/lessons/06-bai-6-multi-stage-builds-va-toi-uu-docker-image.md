<h2 id="1-multi-stage-builds"><strong>1. Multi-stage Builds</strong></h2>
<p>Multi-stage builds allow using multiple FROM statements in a single Dockerfile. Only necessary artifacts are copied to the final image, helping significantly reduce image size.</p>

<h3><strong>Example: Go Application</strong></h3>
<pre><code class="language-dockerfile"># Stage 1: Build
FROM golang:1.22-alpine AS builder
WORKDIR /app
COPY go.mod go.sum ./
RUN go mod download
COPY . .
RUN CGO_ENABLED=0 GOOS=linux go build -o /app/server .
