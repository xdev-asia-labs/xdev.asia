The **Vue.js & Nuxt: From Basics to Advanced** course helps you master Vue.js 3 and Nuxt 3 — progressive frontend/full-stack duos with great developer experience.

## What will you learn?

- **Vue.js Core**: Composition API, Reactivity, Components, Directives, Slots
- **Ecosystem**: Vue Router, Pinia, Composables, VueUse
- **Nuxt 3**: File-based Routing, SSR/SSG, Nitro Server, Data Fetching
- **Full-Stack**: Authentication, Database, File Upload, Real-time, SEO
- **Production**: Testing, Docker, Deployment, Monitoring, Performance

## Request

- Basic knowledge of HTML, CSS, JavaScript ES6+
- Basic TypeScript (recommended)
- Node.js 20+ and npm/pnpm
