The course **React & Next.js: From Basics to Advanced** helps you master React 19+ and Next.js 15+ — the most popular frontend/full-stack duo today, from basic components to production-ready full-stack applications.

## What will you learn?

- **React Core**: Components, Hooks, State Management, Patterns, Performance
- **Next.js**: App Router, Server Components, Server Actions, Data Fetching
- **Full-Stack**: Authentication, Database (Prisma), File Upload, Real-time, Payment
- **Production**: Testing, Docker, CI/CD, Vercel Deployment, Monitoring

## Request

- Basic knowledge of HTML, CSS, JavaScript ES6+
- Basic understanding of TypeScript (recommended)
- Node.js 20+ and npm/pnpm
