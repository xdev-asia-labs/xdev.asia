Khóa học **React & Next.js: Từ Cơ bản đến Nâng cao** giúp bạn làm chủ React 19+ và Next.js 15+ — bộ đôi frontend/full-stack phổ biến nhất hiện nay, từ component cơ bản đến production-ready full-stack application.

## Bạn sẽ học được gì?

- **React Core**: Components, Hooks, State Management, Patterns, Performance
- **Next.js**: App Router, Server Components, Server Actions, Data Fetching
- **Full-Stack**: Authentication, Database (Prisma), File Upload, Real-time, Payment
- **Production**: Testing, Docker, CI/CD, Vercel Deployment, Monitoring

## Yêu cầu

- Kiến thức HTML, CSS, JavaScript ES6+ cơ bản
- Hiểu biết cơ bản về TypeScript (khuyến khích)
- Node.js 20+ và npm/pnpm
