<h2 id="1-django-la-gi"><strong>1. ジャンゴとは何ですか?</strong></h2>

<p>Django は、「バッテリーが付属」した Python Web フレームワークであり、ORM、管理パネル、認証、フォーム処理、テンプレート エンジン、その他多くのモジュールを最初から提供します。</p>

<thead><tr><th>特長</th><th>ジャンゴ</th><th>ファストAPI</th><th>フラスコ</th><th>ララベル</th></tr></thead>
<tbody>
<tr><td>言語</td><td>パイソン</td><td>パイソン</td><td>パイソン</td><td>PHP</td></tr>
<tr><td>種類</td><td>フルスタック</td><td>APIのみ</td><td>マイク</td><td>フルスタック</td></tr>

<tr><td>管理者</td><td>内蔵</td><td>いいえ</td><td>いいえ</td><td>サードパーティ</td></tr>
<tr><td>非同期</td><td>ジャンゴ 5.x</td><td>ネイティブ</td><td>限定</td><td>いいえ</td></tr>
<tr><td>ユースケース</td><td>フルスタック、CMS</td><td>API、マイクロサービス</td><td>シンプルなアプリ</td><td>フルスタック</td></tr>

<h2 id="3-cai-dat"><strong>3. プロジェクトのインストールと作成</strong></h2>

<pre><code class="language-bash"># Python virtual environment
python -m venv .venv
source .venv/bin/activate

# Cài Django
pip install django djangorestframework

# Tạo project
django-admin startproject config .

# Tạo app
python manage.py startapp users
python manage.py startapp products
</code></pre>

<h2 id="4-cau-truc"><strong>4. プロジェクトの構造</strong></h2>

<pre><code class="language-text">myproject/
├── config/
│   ├── __init__.py
│   ├── settings.py      # Configuration
│   ├── urls.py           # Root URL routing
│   ├── wsgi.py           # WSGI entry
│   └── asgi.py           # ASGI entry (async)
├── users/
│   ├── models.py         # Data models
│   ├── views.py          # View logic
│   ├── urls.py           # App URL routing
│   ├── serializers.py    # DRF serializers
│   ├── admin.py          # Admin config
│   └── tests.py          # Tests
├── manage.py
└── requirements.txt
</code></pre>

    'django.contrib.contenttypes',

    'rest_framework',

        'PORT': '5432',

}
