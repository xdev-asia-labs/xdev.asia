**Flutter & Dart: 基本から上級** コースでは、Google の最も人気のあるクロスプラットフォーム UI フレームワークである Flutter を使用して、プロフェッショナルなクロスプラットフォーム (iOS + Android) モバイル アプリを構築できます。

## 何を学ぶのですか?

- **Dart 言語**: Null セーフティ、非同期/待機、ストリーム、パターン マッチング、レコード
- **Flutter UI**: ウィジェット、レイアウト、ナビゲーション、アニメーション、カスタム ペイント
- **状態管理**: Riverpod、Bloc/Cubit、プロバイダー パターン
- **データ**: REST API、Firebase、ローカル ストレージ、オフライン ファースト
- **上級**: プラットフォーム チャネル、プッシュ通知、クリーンなアーキテクチャ
- **本番**: テスト、CI/CD、App Store/Google Play 公開

## リクエスト

- 基本的な OOP プログラミングの知識
- macOS コンピューター (iOS 開発用) または Windows/Linux (Android のみ)
- Flutter SDK 3.27+ および Android Studio / VS Code
